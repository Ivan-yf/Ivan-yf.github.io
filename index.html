八皇后问题(递归回溯实现)

发表于 2020-02-24

wiki：八皇后问题，是一个古老而著名的问题，是回溯算法的典型案例。该问题是国际西洋棋棋手马克斯·贝瑟尔于1848年提出：在8×8格的国际象棋上摆放八个皇后，使其不能互相攻击，即任意两个皇后都不能处于同一行、同一列或同一斜线上，问有多少种摆法。

摆法（暴力）：

  1. 第一个皇后先放在第一行第一列
  2. 第二个皇后放在第二行第一列，然后判断是否跟前面的摆法的皇后冲突，如果冲突，则继续往第二列，第三列摆法，直到找到一个合适的或者达到最后一列
  3. 继续第三个皇后，还是第一列，第二列，直到第八个皇后放在不冲突的位置，就算找到了一个正确解
  4. 当得到一个正确解时，在栈回退到上一个栈时，就会开始回溯，即将第一个皇后放在第一列的所有正确解全部得到
  5. 然后回头继续讲第一个皇后放到第二列，继续循环求解

这里可以采用一维数组即可，queen[8],对应的queen下标代表第几个皇后，对应下标的值表示第几个皇后在第几列。而不用通过说利用二维数组表示8*8棋盘的位置，这样会浪费数组空间。

package recall;

/**
 * 概要
 *
 * @Author Ivan
 * @vesion 1.0
 * @since 0.1.0
 */
public class Queen8 {
    public static void main(String[] args) {
        Queen8 queen = new Queen8();
        queen.way(0);
        System.out.println("总共有" + count + "种解法");
    }
    int queen[] = new int[8];
    static int count = 0;

    /*
     *  每一次递归时，都有一套for循环，当子函数执行完后，会回溯到上一层的for循环中（有点难理解）
     *
     *  @Param [n]
     *  @Return void
     *  @Author Ivan
     */
    private void way(int n){
        if(n == 8){//当达到第九（n=8）个皇后时，即表示已找到正确解，输出并打印，然后回溯到上一层
            print();
            return;
        }
        for(int i = 0;i < queen.length;i++){//每一个皇后都要进行for循环遍历完8列去找合适的位置
            queen[n] = i;//将当前的皇后放在第i个位置
            //判断当前皇后放到i位置时，是否会发生冲突
            if(conflict(n)){
                //如果不冲突，则开始递归，放入第n+1个皇后
                way(n + 1);
            }
            //如果冲突，则接着for循环，将当前皇后的位置放到i+1列的位置
        }
    }
    /*
     * 判断是否当前皇后是否与前面所有已摆放的皇后位置上有冲突
     *
     *  @Param [n]
     *  @Return boolean
     *  @Author Ivan
     */
    private  boolean conflict(int n){
        for(int i = 0;i < n;i++){
            //冲突的条件：与其他皇后在同一列，或者同一斜线
            if(queen[i] == queen[n] || Math.abs(n - i) == Math.abs(queen[n] - queen[i])){
                return false;
            }
        }
        return true;
    }
    private  void print(){
        for(int i = 0;i < queen.length;i++){
            System.out.print(queen[i] + " ");
        }
        count++;
        System.out.println();
    }
}

约瑟夫环

发表于 2020-02-17

约瑟夫（Josephu）环

百度：约瑟夫问题（有时也称为约瑟夫斯置换，是一个出现在计算机科学和数学中的问题。在计算机编程的算法中，类似问题又称为约瑟夫环。又称“丢手绢问题”.）

构建一个单项的环形链表

定义一个节点类

class Boy{
    private int no;//编号
    private Boy next;//指向下一个节点，默认null
    Boy(int no){
        this.no = no;
    }
    public int getNo() {
        return no;
    }

    public void setNo(int no) {
        this.no = no;
    }

    public Boy getNext() {
        return next;
    }

    public void setNext(Boy next) {
        this.next = next;
    }
}
  1. 先创建第一个节点，让first指向该节点，并形成环形；
  2. 后面当我们每创建一个新的节点，就把该节点加入到已有的环形链表中即可.
class CircleLinkList{
//创建一个first节点
    private Boy first = null;
    //添加小孩节点，构建环形的链表
    public void addBoy(int nums){
        if(nums < 1){
            System.out.println("nums的值不正确");
            return;
        }
        Boy curBoy = null;//辅助指针，帮助构建环形链表
        for(int i = 1;i <= nums;i++){
            //根据编号创建小孩节点
            Boy boy = new Boy(i);
            if(i == 1){
                first = boy;
                first.setNext(first) ;//构成环
                curBoy = first;//让curBoy指向第一个节点
            }
            else{
                curBoy.setNext(boy);//将上一个节点与当前新生成的节点相连
                boy.setNext(first);//将当前新生成的节点与头结点相连形成环
                curBoy = boy;//将curBoy节点指向新生的节点
            }
        }
    }
}

遍历环形链表

  1. 先让一个辅助指针（变量）curBoy，指向first节点；
  2. 然后通过一个while循环遍历该环形链表，当curBoy.next==first结束
class CircleLinkList{
//创建一个first节点
    private Boy first = null;
    //遍历环形链表
    public void show(){
        if(first == null){
            System.out.println("链表为空");
            return;
        }
        Boy curBoy = first;
        while(true){
            System.out.println(curBoy.getNo());
            //判断是否到最后
            if(curBoy.getNext() == first){
                break;
            }
            curBoy = curBoy.getNext();
        }
    }
}

环形出圈

根据用户的输入，生成一个小孩出圈的顺序

n=5，即有5个人

k=1，从第一个人开始报数

m=2，数2下然后出圈

  1. 需求创建一个辅助指针（变量）helper，事先应该指向环形链表的最后这个节点小孩报数前，先让first跟helper移动k-1次
  2. 当小孩报数时，让first和helper指针同时移动m-1次
  3. 这时就可以让first指向的小孩节点出圈

first = first.next

helper.next = first

原来first指向的节点就没有任何引用==出圈回收

class CircleLinkList{
    //创建一个first节点
    private Boy first = null;

    //根据用户的输入，计算出圈的顺序
    /**
     *
     * @param startNo 初始在第几个小孩开始数数
     * @param countNo 数多少下出圈
     * @param nums 总共多少个小孩
     */
    public void countBoy(int startNo,int countNo,int nums){
        if(first == null || startNo < 1 || startNo > nums || countNo < 1){
            System.out.println("参数有误");
            return;
        }
        Boy helper = first;
        //将辅助指针指向最后一个节点
        while(true){
            if(helper.getNext() == first){
                break;
            }
            helper = helper.getNext();
        }
        //报数前先让两个节点移动到要报数的位置
        for(int i = 1;i < startNo;i++){
            helper = helper.getNext();
            first = first.getNext();
        }
        //让first和helper指针同时移动countNo-1次,然后出圈
        while (true){
            if(helper == first){
                break;
            }
            for(int i = 0;i < countNo - 1;i++){
                first = first.getNext();
                helper = helper.getNext();
            }
            System.out.println("小孩"+first.getNo()+"出圈");
            first = first.getNext();//将first指向出圈小孩的下一个节点
            helper.setNext(first);//将helper的与出圈小孩的下一个节点相连，重新形成环形
        }
        System.out.println("最后留在圈中的是"+first.getNo());
    }
}
